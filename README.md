# Asian Food Explorer

A final web application project built with HTML, CSS, and vanilla JavaScript.

## Features
- Dish search with TheMealDB API
- Category browsing
- Random dish suggestion
- Meal detail view
- Favorites saved in local storage
- Location search with OpenStreetMap Nominatim API
- Contact form validation
- Responsive design and CSS animations

- index.html
